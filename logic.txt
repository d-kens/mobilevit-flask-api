Apps Logic (How it work)
1. Client sends an image to get it class class label
2. The image is
    - Processed 
    - Fed to the MobileViT model for classification
3. The classification result is saved to the database together with the predicted class
4. The classification result is returned to the user.

Logic Implementation

   - get the image
